Help & how to use this course

V3.1 (July 2026)

New to the topic? Take the modules in order, starting at PQC-001. Stage 1 assumes no background at all.

Already technical? You can jump to Stage 2 (standards) or Stage 3 (build & migrate).

  • Each module has plain-English lessons, interactive diagrams, worked examples with real numbers, "Go deeper" expandable detail, hands-on labs, and a scored quiz with explanations.
  • Pass a module's quiz (70%+) and it is marked complete on this page. Progress is stored only in your browser.
  • Use the Glossary anytime for plain definitions of key terms.
  • Toggle dark/light mode and reopen Help from any page.

This is educational content, not production cryptographic approval.
