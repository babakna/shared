How to Use This PQC Lab

Start with PQC-101 if you are new to quantum-safe security. Use the risk triage tool to decide what systems need attention first, then use the roadmap cards to pick deeper modules as they are added.

  • Use light/dark mode from the top bar.
  • Open PQC-101 for the live interactive course.
  • Placeholder modules show the planned expansion path.
  • Version labels use month-only review markers, not exact dates.
