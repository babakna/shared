4. Scenario lab: enterprise migration board

Run a decision meeting. Each system needs an owner, inventory entry, risk tier, vendor status, target architecture, test result, rollback plan, and exception decision.

Decision inputs

Data lifetime, exposure, criticality, dependency, test evidence.

Decision outputs

Migrate, pilot, wait for vendor, exception, or retire.

Audit record

Who approved, why, evidence, review date window.

Feedback loop

Update inventory and roadmap as standards and products change.
